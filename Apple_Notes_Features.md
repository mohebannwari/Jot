# Apple Notes -- Complete Feature List (as of iOS 26 / macOS 26)

---

## Text & Formatting

- Rich text formatting -- bold, italic, underline, strikethrough
- Headings -- Title, Heading, Subheading styles
- Lists -- bulleted, numbered, and checklists (with auto-sorting of checked items)
- Tables -- add/remove/rearrange rows and columns
- Collapsible sections -- tap a heading to collapse everything beneath it (iOS 18+)
- Markdown import/export -- iOS 26+ / macOS 26+. Import `.md` files via share sheet, export from within a note.
- Inline links -- add hyperlinks to text

## Math & Calculations

- Math Notes -- type or handwrite equations, auto-solved when you add `=` (iOS 18+)
- Variables -- assign and reuse values across equations
- Unit conversions -- currency, time, distance, weight, etc.
- Graphing -- auto-generated graphs for applicable equations

## Audio

- Audio recording -- record directly inside a note (iPhone 12+, iOS 18+)
- Live transcription -- recordings auto-transcribed in real time (English only)
- Phone call recording -- record calls from the Phone app, saved + transcribed in Notes (iOS 26+)
- Audio summaries -- Apple Intelligence can summarize transcriptions (18.1+)

## Drawing & Handwriting

- Freeform drawing -- Apple Pencil or finger, multiple pen/pencil/highlighter tools
- Handwriting recognition -- converts handwritten text to typed text
- Reed pen tool -- calligraphy-style pen (iOS 26+)
- Ruler tool -- draw straight lines
- Shape recognition -- auto-corrects hand-drawn shapes

## Attachments & Media

- Photos & videos -- embed inline
- Document scanner -- built-in camera-based scanner for receipts, contracts, etc.
- PDF viewing -- inline PDF preview
- Web links -- link previews with thumbnails
- Maps -- embed location pins
- File attachments -- various file types
- Drag and drop -- images, files into notes

## Organization

- Folders & subfolders -- nested hierarchy
- Smart Folders -- auto-populate based on filters (tags, mentions, dates, checklists, attachments, etc.)
- Tags -- `#hashtag` anywhere in a note, with a Tag Browser
- Pinned notes -- stick important notes to the top
- Sorting -- by date edited, date created, title
- Attachments browser -- view all attachments across notes in one gallery

## Search

- Full-text search -- across all notes
- Handwriting search -- searches inside handwritten content
- Scanned document search -- OCR on scanned docs
- Spotlight integration -- find notes from system-wide search
- In-note Find -- search within a single note (iOS 18+)

## Security & Privacy

- Note locking -- per-note lock with device passcode, Face ID, or Touch ID
- iCloud Keychain sync -- lock key syncs across devices
- End-to-end encryption -- for locked notes via Advanced Data Protection
- Limitation: locked notes cannot be collaborated on

## Collaboration

- Real-time collaboration -- up to 100 people per note
- Share via Messages, Mail, or link
- Per-note permissions -- view-only or edit access
- Shared folders -- collaborate on entire folder hierarchies
- Mentions -- `@name` to notify collaborators
- Activity view -- see who changed what and when

## Apple Intelligence (18.1+ / macOS 15.1+)

- Writing Tools -- proofread, rewrite, summarize, change tone
- Visual Intelligence -- auto-categorizes notes (receipts, to-dos, meeting notes, etc.)
- Smart search suggestions
- Action recommendations -- suggests creating reminders from to-do items, extracting dates/addresses

## Cross-Platform & System Integration

- iCloud sync -- across iPhone, iPad, Mac
- Siri integration -- "Hey Siri, create a note"
- Spotlight indexing -- content-level search
- Shortcuts support -- automation via Shortcuts app
- Share sheet -- send content to Notes from any app
- Quick Notes -- system-wide hot corner / shortcut to jot something down
- Apple Watch -- basic note viewing + dictation (iOS 26+)
- Widgets -- home screen / lock screen widgets

## Design (iOS 26+ / macOS 26+)

- Liquid Glass UI -- translucent, modern interface redesign

---

## What Apple Notes Does NOT Have

- No slash command palette
- No syntax-highlighted code blocks
- No split view / side-by-side editing
- No version history (user-facing)
- No inter-note linking (until iOS 17 added basic `Add Link`, but it's buried -- not `@mention` style)
- No local backup/restore (relies entirely on iCloud)
- No font/theme customization (you get what Apple gives you)
- No bulk export (individual Markdown export only as of iOS 26)
- No embedded web clip previews (just link thumbnails)
